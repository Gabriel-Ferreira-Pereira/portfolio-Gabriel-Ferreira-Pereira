# Portfólio — Gabriel Ferreira Pereira

Portfólio pessoal desenvolvido como atividade prática da disciplina **Fundamentos da Programação Web** do curso de Análise e Desenvolvimento de Sistemas — UNINTER.

## 📋 Sobre o projeto

Site do tipo *Single Page* com navegação por âncoras, contendo quatro seções obrigatórias:

- **Sobre mim** — apresentação pessoal, foto e hobbies
- **Formação** — educação, cursos e idiomas
- **Portfólio** — projetos desenvolvidos com links
- **Contato** — formulário com validação em JavaScript

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (sem frameworks)
- JavaScript puro (sem bibliotecas)

## ⚙️ Funcionalidades

- Tema claro/escuro com preferência salva no `localStorage`
- Menu hambúrguer responsivo para dispositivos móveis
- Validação de formulário com regex de e-mail
- Modal de confirmação ao enviar o formulário
